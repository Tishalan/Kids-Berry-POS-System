# Kids Berry Management System

A multi-branch management system for Kids Berry, built with PHP.

## Directory Structure

- `/admin/` - Administrative control panel and configuration.
- `/branch_chunnakam/` - Management files for the Chunnakam branch (Mega Centre).
- `/cashier/` - Cashier portal for the Jaffna branch.
- `/stock_keeper/` - Stock Keeper portal for the Jaffna branch.
- `/assets/` - Static assets including images, css, and js.

## Getting Started

1. Set up a local PHP server (e.g. XAMPP, WAMP).
2. Clone this repository into your `htdocs` or `www` directory.
3. Import the required database schemas into MySQL.
4. Update database configuration credentials in the respective PHP files if running in a different environment.
5. Navigate to `index.php` in your browser to access the branch selection page.
